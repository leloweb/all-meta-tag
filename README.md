## Create Custom Meta Tags

Use custom meta tags to store data that you need in javascript, instead of hard-coding that data into your javascript.  Here's some examples:

``` html
<meta name="google-analytics" content=""/>
<meta name="disqus" content=""/>
<meta name="uservoice" content=""/>
<meta name="mixpanel" content=""/>
```
